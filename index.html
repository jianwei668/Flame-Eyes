📋 游戏规则 —《神话宇宙》
【卡牌分类】 • 神魔卡 — 战斗单位，有战力值和生命值，降临到人界参与战斗 • 法宝卡 — 装备到人界的神魔卡，增强属性 • 法术卡 — 给人界的神魔卡使用，产生各种效果 • 圣地卡 — 放置到人界，不会被攻击破坏 【祭品系统】 需要祭品的神魔卡降临到人界时，必须将卡上记载数量的神魔卡从自己人界送去地狱作为祭品，才能降临。 0祭品可直接降临，1~5祭品需消耗对应数量。 【对战区域】 • 手牌 — 手中持有的卡牌，上限6张 • 人界前排 — 可发动攻击的战斗区域 • 人界后排 — 后排神魔卡不能发动攻击，前排有卡时后排不会被攻击 • 天堂 — 累积到13张神魔卡即胜利 • 地狱 — 累积到13张神魔卡即败北 • 卡组区 — 抽卡来源，无卡可抽时判定败北 【对战流程】 1. 剪刀石头布决定先后手 2. 每回合开始抽卡，最多补满6张手牌 3. 将神魔卡降临到人界（前排/后排），需支付祭品 4. 用人界前排神魔卡攻击对方神魔卡 5. 攻击时战力值差额从对方生命值扣除 6. 生命值归0的神魔卡被破坏送入地狱 7. 回合结束手牌超过6张须舍弃多余卡牌至地狱 【战斗规则】 • 只能攻击对方前排神魔卡，前排无卡才能攻击后排 • 战力值相同双方不扣生命值 • 战力差额 = 从被攻击方生命值扣除的数值 • 先手方第1回合不能发动攻击 【法宝卡规则】 • 装备到人界神魔卡使用 • 同名法宝卡不能重复装备到同一张神魔卡 • 祭品降临时法宝卡可被继承 【法术卡规则】 • 必须给人界的神魔卡使用才能发动 • 同名法术卡在同一回合不能重复用于同一张神魔卡 • 用完的法术卡送入地狱 【胜负判定】 • 天堂累积13张神魔卡 → 胜利 • 地狱累积13张神魔卡 → 败北 • 无卡可抽 → 败北 • 进行最后一步获胜操作前须告知对方："你已经死了"（类似象棋将军） 【操作指引】 • 单击手牌 → 降临到人界前排（需祭品时自动扣除） • 右键手牌 → 降临到人界后排 • 单击人界卡牌 → 送入地狱（破坏） • 右键前排卡 → 移到后排 / 右键后排卡 → 移到前排 • 右键地狱卡 → 送入天堂 • 双击任意卡牌 → 查看详细信息 完整规则及全部法术效果请在页面内查看。
